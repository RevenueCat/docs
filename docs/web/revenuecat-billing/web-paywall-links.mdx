## How it works

1. Set up an [Offering](/tools/offerings-guide)
2. Add some [RC Billing products](/web/revenuecat-billing/product-setup) to your offering
3. Visit your Offering page and click on the Web Paywall Link tab
4. Finally click on the Create a Web Paywall Link for this offering
